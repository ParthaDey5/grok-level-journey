#Deepseek Theme

A customized theme for Windsurf/VS Code with enhanced colors and better contrast.

## Installation

### Method 1: Install from VSIX
1. Build the extension: `vsce package`
2. Install the `.vsix` file in Windsurf/VS Code

### Method 2: Install from Source
1. Clone this repository
2. Open in Windsurf/VS Code
3. Press `F5` to run in debug mode

## Features

- Enhanced syntax highlighting
- Better contrast ratios
- Improved readability
- Custom color palette optimized for coding
- Semantic token highlighting support


## Customization

You can customize the theme by editing the `themes/Custom-Dark-Plus.json` file.

## Development

This theme is built using the VS Code Extension API. For more information on creating themes, see the [VS Code Theme Documentation](https://code.visualstudio.com/api/extension-guides/color-theme).
